# 🏘️ Condomínio Garden - Mapeamento ACS

Uma aplicação web simples, rápida e responsiva desenvolvida para auxiliar Agentes Comunitários de Saúde (ACS) na gestão e atualização de cadastros de moradores em condomínios.

## 🎯 O Problema
O trabalho de campo do ACS envolve mapear centenas de residências, lidando frequentemente com pranchetas, papéis e anotações manuais. Isso dificulta a visualização geral do progresso e o controle de quais moradores estavam ausentes ou pendentes de atualização.

## 💡 A Solução
Este projeto substitui a prancheta de papel por uma interface mobile-first, permitindo que o Agente registre o status de cada apartamento com poucos toques na tela do celular.

### ✨ Funcionalidades
- **Interface Mobile-First:** Pensada para ser operada com uma mão no celular durante as visitas.
- **Armazenamento Local (Offline):** Os dados são salvos no `localStorage` do navegador, permitindo o uso mesmo sem internet no condomínio.
- **Filtros e Buscas Racionadas:** Busca instantânea por bloco, apartamento ou observações.
- **Controle de Status:** Separação visual por cores (Pendente, Concluído, Ausente).
- **Barra de Progresso:** Feedback visual e em tempo real do avanço do mapeamento do condomínio.
- **Notas Personalizadas:** Espaço para adicionar observações a cada unidade (ex: "Morador viaja muito", "Cachorro bravo").
